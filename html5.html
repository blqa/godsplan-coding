¿Qué rayos es HTML?

Imagina que vas a construir una casa. Antes de pintar las paredes (CSS) o ponerle electricidad (JavaScript), necesitas los cimientos, las vigas y las paredes de ladrillo.

Eso es HTML (HyperText Markup Language). Es el esqueleto de cualquier página web. No se encarga de que se vea "bonito", se encarga de que exista.

Dato Curioso

HTML no es un lenguaje de programación (no hace cálculos). Es un lenguaje de marcado. Usamos "etiquetas" para decirle al navegador qué es cada cosa.

La Anatomía de una Etiqueta

Casi todo en HTML funciona como un sándwich. Tienes una etiqueta de apertura, el contenido en medio, y una etiqueta de cierre.

<p> Hola, soy un párrafo </p>
Apertura Contenido Cierre
index.html
<!-- Esto es un comentario -->
<h1>Hola Mundo</h1>
<p>Este es mi primer sitio web.</p>
